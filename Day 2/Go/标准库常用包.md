Go 语言的标准库非常丰富，涵盖了网络、系统、编码、并发等各个方面。下面介绍您提到的几个常用标准库包：`fmt`、`os`、`io`、`net/http`、`encoding/json` 和 `sync`。

---

## 1. fmt 包

**用途**：实现格式化输入输出，类似于 C 语言的 `printf` 和 `scanf`。

**常用功能**：

- **输出函数**：
    - `Println`：打印并换行
    - `Printf`：格式化打印
    - `Print`：直接打印  
      示例：
  ```go
  name := "Go"
  age := 10
  fmt.Println("Hello,", name)          // Hello, Go
  fmt.Printf("Age: %d years\n", age)    // Age: 10 years
  ```

- **输入函数**：
    - `Scanln`、`Scanf`、`Scan` 从标准输入读取  
      示例：
  ```go
  var input string
  fmt.Scanln(&input)   // 读取一行输入
  ```

- **字符串格式化**：
    - `Sprintf` 返回格式化后的字符串，而不打印  
      示例：
  ```go
  s := fmt.Sprintf("value: %d", 42)
  ```

---

## 2. os 包

**用途**：提供与操作系统交互的功能，如文件操作、环境变量、进程管理等。

**常用功能**：

- **文件操作**：
    - `os.Open` 打开文件（只读）
    - `os.Create` 创建文件
    - `os.Remove` 删除文件  
      示例：
  ```go
  file, err := os.Open("test.txt")
  if err != nil {
      log.Fatal(err)
  }
  defer file.Close()
  ```

- **环境变量**：
    - `os.Getenv` 获取环境变量
    - `os.Setenv` 设置环境变量

- **命令行参数**：
    - `os.Args` 获取命令行参数切片

- **进程相关**：
    - `os.Exit` 退出程序
    - `os.Getpid` 获取当前进程 ID

---

## 3. io 包

**用途**：提供基本的 I/O 接口，如 `Reader` 和 `Writer`，以及一些通用的操作函数。

**核心接口**：

- **`io.Reader`**：表示可读取的流，方法 `Read(p []byte) (n int, err error)`
- **`io.Writer`**：表示可写入的流，方法 `Write(p []byte) (n int, err error)`

**常用函数**：

- `io.Copy`：从 `Reader` 复制到 `Writer`，直到 EOF  
  示例：
  ```go
  r := strings.NewReader("hello")
  io.Copy(os.Stdout, r)   // 输出 hello
  ```

- `io.ReadAll`：一次性读取所有数据（Go 1.16+ 推荐使用 `io.ReadAll` 替代 `ioutil.ReadAll`）

- **组合接口**：
    - `io.ReadWriter`：同时实现 `Reader` 和 `Writer`
    - `io.Closer`：关闭接口

---

## 4. net/http 包

**用途**：提供 HTTP 客户端和服务端的实现。

**服务端**：

- **路由与处理**：
    - `http.HandleFunc` 注册路由处理函数
    - `http.ListenAndServe` 启动 HTTP 服务  
      示例：
  ```go
  http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
      fmt.Fprintf(w, "Hello, you've requested: %s\n", r.URL.Path)
  })
  http.ListenAndServe(":8080", nil)
  ```

**客户端**：

- **发送请求**：
    - `http.Get` 发起 GET 请求
    - `http.Post` 发起 POST 请求
    - 更灵活的使用 `http.Client`  
      示例：
  ```go
  resp, err := http.Get("http://example.com")
  if err != nil { /* 处理错误 */ }
  defer resp.Body.Close()
  body, _ := io.ReadAll(resp.Body)
  fmt.Println(string(body))
  ```

- **常用类型**：`http.Request`、`http.ResponseWriter`、`http.Client` 等。

## 5. encoding/json 包

**用途**：实现 JSON 数据的编码（Marshal）和解码（Unmarshal）。

**常用函数**：

- **`json.Marshal`**：将 Go 值编码为 JSON 字节切片
- **`json.Unmarshal`**：将 JSON 数据解析到 Go 值中  
  示例：
  ```go
  type Person struct {
      Name string `json:"name"`
      Age  int    `json:"age"`
  }

  p := Person{Name: "Alice", Age: 30}
  data, _ := json.Marshal(p)
  fmt.Println(string(data)) // {"name":"Alice","age":30}

  var p2 Person
  json.Unmarshal(data, &p2)
  fmt.Println(p2.Name) // Alice
  ```

- **流式处理**：
    - `json.NewEncoder` 写入到 `io.Writer`
    - `json.NewDecoder` 从 `io.Reader` 读取  
      示例：
  ```go
  json.NewEncoder(os.Stdout).Encode(p)
  ```

- **结构体标签（Tag）**：用于自定义 JSON 字段名，如 `json:"name,omitempty"`。

---

## 6. sync 包

**用途**：提供基本的并发同步原语，用于协调 goroutine 的访问。

**常用类型**：

- **`sync.Mutex`**：互斥锁，用于保护临界区  
  示例：
  ```go
  var mu sync.Mutex
  var counter int

  func increment() {
      mu.Lock()
      counter++
      mu.Unlock()
  }
  ```

- **`sync.RWMutex`**：读写锁，允许多个读或一个写

- **`sync.WaitGroup`**：等待一组 goroutine 完成  
  示例：
  ```go
  var wg sync.WaitGroup
  for i := 0; i < 5; i++ {
      wg.Add(1)
      go func(i int) {
          defer wg.Done()
          fmt.Println("goroutine", i)
      }(i)
  }
  wg.Wait() // 等待所有 goroutine 结束
  ```

- **`sync.Once`**：确保某个函数只执行一次（如单例初始化）
- **`sync.Map`**：并发安全的 map（适合读多写少的场景）
- **`sync.Cond`**：条件变量，用于等待或通知事件

## 总结

这些标准库包覆盖了日常开发中最基础、最常用的功能：

- `fmt`：格式化 I/O
- `os`：操作系统接口
- `io`：I/O 抽象接口
- `net/http`：HTTP 通信
- `encoding/json`：JSON 处理
- `sync`：并发同步

熟练掌握它们可以帮助你快速构建可靠、高效的 Go 程序。
