
## 逐行解释 Go 代码

```go
package main
```
声明当前文件属于 `main` 包，入口程序必须使用 `main` 包。

```go
import (
	"bytes"
	"fmt"
	"io"
	"os"
	"strings"
)
```
导入需要的标准库：
- `bytes` - 操作字节切片和缓冲区
- `fmt` - 格式化输入输出
- `io` - I/O 操作的工具函数
- `os` - 操作系统功能（文件操作、标准输入输出）
- `strings` - 字符串处理

```go
func main() {
```
程序入口函数。

```go
	// 1. 使用 Reader 和 Writer
	r := strings.NewReader("Hello, Go!")
```
创建一个字符串 Reader，将字符串转换为可读取的数据源。

```go
	var buf bytes.Buffer
```
声明一个 `bytes.Buffer` 变量，用于临时存储数据（可增长的字节切片）。

```go
	n, err := io.Copy(&buf, r)
```
调用 `io.Copy`：
- 第一个参数是 **Writer**（目标）：`&buf` 是 Buffer 的指针
- 第二个参数是 **Reader**（源）：`r` 是字符串Reader
- 将源数据复制到目标，返回复制的字节数 `n` 和可能的错误 `err`

```go
	if err != nil {
		fmt.Println("copy error:", err)
		return
	}
```
如果复制过程中发生错误，打印错误信息并退出函数。

```go
	fmt.Printf("copied %d bytes: %s\n", n, buf.String())
```
打印复制的字节数和 Buffer 中的内容（转换为字符串）。

```go
	// 2. 使用 MultiWriter 同时写入文件和控制台
	file, _ := os.Create("test.txt")
```
创建（或覆盖）一个名为 `test.txt` 的文件，返回文件对象和错误（这里用 `_` 忽略了错误）。

```go
	defer file.Close()
```
使用 `defer` 确保函数结束时关闭文件，释放资源。

```go
	mw := io.MultiWriter(os.Stdout, file)
```
创建一个 **MultiWriter**，它会将所有写入操作同时发送给多个 Writer：
- `os.Stdout` - 标准输出（控制台）
- `file` - 刚才创建的文件

```go
	io.WriteString(mw, "This goes to both stdout and file.\n")
```
将字符串写入 `MultiWriter`，这条消息会**同时**显示在控制台和写入文件。

```go
	// 3. 使用 LimitReader 限制读取长度
	limited := io.LimitReader(strings.NewReader("1234567890"), 5)
```
创建一个 **LimitReader**，限制最多只能读取 5 个字节：
- 原始数据："1234567890"（10个字符）
- 限制：最多读取 5 字节
- 实际读取："12345"

```go
	data, _ := io.ReadAll(limited)
```
读取 LimitReader 中的所有剩余数据。

```go
	fmt.Printf("limited read: %s\n", data) // 输出 "12345"
```
打印限制读取的结果。

```go
	// 4. 使用 TeeReader 同时读取并写入
	var teeBuf bytes.Buffer
```
再创建一个 Buffer，用于存储"分流"的数据。

```go
	tee := io.TeeReader(strings.NewReader("tee example"), &teeBuf)
```
创建 **TeeReader**：
- 第一个参数：原始数据源 "tee example"
- 第二个参数：写入目标 `&teeBuf`
- 读取时数据会**原样复制**一份到 teeBuf

```go
	io.Copy(os.Stdout, tee) // 输出到 stdout，同时 teeBuf 也会得到数据
```
将 TeeReader 的内容复制到标准输出，同时数据会自动写入 teeBuf。

```go
	fmt.Printf("\ncaptured in buffer: %s\n", teeBuf.String())
```
打印 teeBuf 中捕获的内容。

```go
}
```
函数结束。
