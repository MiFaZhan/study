# 二分查找（Binary Search）

## 概念
二分查找是一种高效的查找算法，通过不断将查找区间对半分割，快速定位目标元素。

## 前提条件
⚠️ **数组必须是有序的**（升序或降序）

## 算法思想
1. 找到数组的中间元素
2. 如果中间元素等于目标值，返回索引
3. 如果中间元素小于目标值，在右半部分继续查找
4. 如果中间元素大于目标值，在左半部分继续查找
5. 重复以上步骤，直到找到目标或查找区间为空

## 时间复杂度
- **最好情况**：O(1) - 中间元素就是目标
- **最坏情况**：O(log n) - 需要不断二分
- **平均情况**：O(log n)

## 空间复杂度
- **迭代版本**：O(1)
- **递归版本**：O(log n) - 递归调用栈

## 实现方式

### 1. 迭代版本（推荐）
```go
func binarySearch(arr []int, target int) int {
    left := 0
    right := len(arr) - 1
    
    for left <= right {
        mid := left + (right-left)/2  // 防止溢出
        
        if arr[mid] == target {
            return mid
        } else if arr[mid] < target {
            left = mid + 1
        } else {
            right = mid - 1
        }
    }
    return -1
}
```

### 2. 递归版本
```go
func binarySearchRecursive(arr []int, target, left, right int) int {
    if left > right {
        return -1
    }
    
    mid := left + (right-left)/2
    
    if arr[mid] == target {
        return mid
    } else if arr[mid] < target {
        return binarySearchRecursive(arr, target, mid+1, right)
    } else {
        return binarySearchRecursive(arr, target, left, mid-1)
    }
}
```

## 关键点

### 1. 中点计算
```go
// ❌ 可能溢出
mid := (left + right) / 2

// ✅ 推荐写法
mid := left + (right-left)/2
```

### 2. 边界条件
- 循环条件：`left <= right`（注意等号）
- 更新边界：`left = mid + 1` 或 `right = mid - 1`

### 3. 处理重复元素
查找第一个出现的位置：
```go
if arr[mid] == target {
    result = mid
    right = mid - 1  // 继续在左半部分查找
}
```

查找最后一个出现的位置：
```go
if arr[mid] == target {
    result = mid
    left = mid + 1  // 继续在右半部分查找
}
```

## 适用场景
1. 数据量大
2. 数组已排序
3. 需要频繁查找
4. 对查找效率要求高

## 优缺点

### 优点
- 效率高，时间复杂度为 O(log n)
- 适合大数据集

### 缺点
- 要求数据必须有序
- 只适用于支持随机访问的数据结构（如数组）
- 不适用于链表

## 查找过程示例
在数组 `[1, 3, 5, 7, 9, 11, 13, 15]` 中查找 `7`：

```
步骤1: left=0, right=7, mid=3, arr[mid]=7 ✓ 找到！
```

在数组 `[1, 3, 5, 7, 9, 11, 13, 15]` 中查找 `13`：

```
步骤1: left=0, right=7, mid=3, arr[mid]=7  → 目标在右半部分
步骤2: left=4, right=7, mid=5, arr[mid]=11 → 目标在右半部分
步骤3: left=6, right=7, mid=6, arr[mid]=13 ✓ 找到！
```

## 变体应用
1. 查找第一个等于目标值的位置
2. 查找最后一个等于目标值的位置
3. 查找第一个大于等于目标值的位置
4. 查找最后一个小于等于目标值的位置
5. 旋转数组中的查找
