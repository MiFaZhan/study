



## 二叉搜索树（BST）是什么？

**二叉搜索树（Binary Search Tree，简称 BST）** 是一种特殊的二叉树，它具有以下重要特性：

### 核心规则

对于 BST 中的**每个节点**：
1. **左子树**中所有节点的值都**小于**当前节点的值
2. **右子树**中所有节点的值都**大于**当前节点的值
3. 左右子树本身也必须是 BST

### 示例图示

```
        4
       / \
      2   6
     / \ / \
    1  3 5  7
```

在这个 BST 中：
- 节点 4 的左边（1,2,3）都小于 4
- 节点 4 的右边（5,6,7）都大于 4
- 每个子树都遵循相同规则

### BST 的优势

**1. 查找效率高**
- 查找某个值时，每次都能排除一半的节点
- 平均时间复杂度：O(log n)
- 最坏情况（退化为链表）：O(n)

**2. 中序遍历得到有序序列**
- 对 BST 进行中序遍历，会得到升序排列的值
- 这就是为什么程序中 BST 的中序遍历输出：`1 2 3 4 5 6 7`

### 代码中的体现

在您的代码中，[insertBST](file:///D:\Project\study\Day 5\02-数据结构与算法\BinaryTree\main.go#L57-L70) 函数就体现了 BST 的构建规则：

```go
if val < root.Val {
    root.Left = insertBST(root.Left, val)  // 小的放左边
} else if val > root.Val {
    root.Right = insertBST(root.Right, val)  // 大的放右边
}
```

### 实际应用

- 数据库索引
- 字典实现
- 动态集合的维护
- 范围查询

BST 是数据结构中非常重要的概念，它将数据的有序性和树形结构结合在一起，提供了高效的查找、插入和删除操作。
