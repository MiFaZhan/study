
`right...` 中的三个点 `...` 是 Go 语言的**展开操作符**（spread operator）。

**作用：**

将切片展开为多个独立的参数。

**为什么需要？**

`append()` 函数的签名是：
```go
func append(slice []Type, elems ...Type) []Type
```

- 第一个参数：目标切片
- 第二个参数：可变参数，可以接受多个值

**对比示例：**

```go
// 假设 right = [6, 12]

// 不使用 ... (错误)
append(left, right)  
// 结果: left + [6, 12] 作为一个整体
// 即: [1, 2, 3, 5, [6, 12]] ❌

// 使用 ... (正确)
append(left, right...)  
// 结果: left + 6 + 12
// 即: [1, 2, 3, 5, 6, 12] ✅
```

**实际效果：**

`right...` 相当于将切片 `[6, 12]` 展开为 `6, 12` 两个独立的参数，这样 `append` 就能正确地将每个元素逐个添加到目标切片中。

在您的代码中，这行：
```go
return append(append(left, pivot), right...)
```

实际上等价于：
```go
temp := append(left, pivot)      // temp = [1, 2, 3, 5]
return append(temp, 6, 12)       // 结果 = [1, 2, 3, 5, 6, 12]
```

这就是为什么需要 `...` 来展开切片的原因。
