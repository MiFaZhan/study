
## CSS Grid 布局功能解释

这是一个 **CSS Grid（网格布局）** 的典型应用，用于创建二维页面布局。

### 核心属性解析：

| 属性 | 值 | 功能 |
|------|-----|------|
| `display: grid;` | - | 将元素设置为网格容器 |
| `grid-template-columns` | `200px 1fr 200px` | 定义 **3列**：左200px、右200px、中间自适应 |
| `grid-template-rows` | `80px auto 60px` | 定义 **3行**：首行80px、末行60px、中间自动 |
| `gap` | `10px` | 网格项之间的间距（行和列） |
| `min-height: 100vh` | - | 最小高度占满整个视口 |

### 布局结构图示：

```
┌─────────────────────────────────────┐
│           Header (header)           │  ← grid-column: 1/4 (跨3列)
├──────────┬──────────┬───────────────┤
│          │          │               │
│  200px   │  1fr     │    200px      │  ← 行高: 80px
│          │  (自适应) │               │
├──────────┼──────────┼───────────────┤
│          │          │               │
│  内容区   │  内容区   │    内容区      │  ← 行高: auto (自动)
│          │          │               │
├──────────┼──────────┼───────────────┤
│          │          │               │
│  200px   │  1fr     │    200px      │  ← 行高: 60px
│          │  (自适应) │               │
└──────────┴──────────┴───────────────┘
```

### `fr` 单位说明：

- `1fr` 表示 **等分剩余空间**
- 假设容器宽度 1000px，列定义为 `200px 1fr 200px`
- 剩余空间 = 1000 - 200 - 200 = 600px
- 中间列宽度 = 600px

### 适用场景：

- ✅ 页面整体布局（头部、侧边栏、主内容、底部）
- ✅ 相册、图片画廊
- ✅ 仪表盘界面
- ❌ 一维排列（用 Flexbox 更简单）

这个布局非常适合做经典的 **"左-中-右" 三栏布局** 或 **圣杯布局**。

## grid-column 样式功能解释

这是 CSS Grid 中的 **网格列定位** 属性。

### 核心属性：`grid-column: 1 / 4;`

| 语法 | 含义 |
|------|------|
| `grid-column: 1 / 4;` | 让元素占据 **第1列到第4列**（即跨越整个网格宽度） |

### 详细说明：

- `1` 表示 **起始列**（从第1列开始）
- `4` 表示 **结束列**（到第4列之前结束）
- 相当于跨越了 **3列**（列号1、2、3）

### 布局效果图示：

```
┌─────────────────────────────────────┐
│         HEADER (跨3列)              │  ← grid-column: 1/4
├──────────┬──────────┬───────────────┤
│  col-1   │  col-2   │    col-3      │
│          │          │               │
├──────────┴──────────┴───────────────┤
│         ...其他内容...              │
└─────────────────────────────────────┘
```

### 对比其他写法：

```css
/* 两种写法效果相同 */
grid-column: 1 / 4;    /* 简写形式 */
grid-column: 1 / span 3; /* 使用 span 关键字，跨3列 */
grid-column-start: 1; 
grid-column-end: 4;   /* 拆开写 */
```

### 常用场景：

- 页面头部（header）横跨整行
- 页面底部（footer）横跨整行
- 侧边栏（sidebar）跨越多行

结合 `background: skyblue;`，这个 `.header` 类通常用于设置页面顶部的 **蓝色导航栏/标题区域**。
