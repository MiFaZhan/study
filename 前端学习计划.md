# 🌟 前端开发学习路线图
## 一、HTML
### 1. 基本标签使用
- 学习基本标签：`<html>`、`<head>`、`<body>`、`<h1>`-`<h6>`、`<p>`、`<a>`、`<img>`、`<ul>`/`<ol>`、`<div>`/`<span>`
### 2. 表单与交互
- 学习 `<form>`、`<input>`、`<textarea>`、`<button>` 等表单标签
---
## 二、CSS
> **核心思想**：CSS 控制网页样式，实现内容与样式的分离
### 1. 核心概念与选择器
- 学习选择器：元素选择器、类选择器 (`.class`)、ID 选择器 (`#id`)
- 学习样式属性：颜色、字体、边框等
### 2. 盒模型与布局
- 理解 `content`、`padding`、`border`、`margin` 的关系
- 使用 `display` 属性（`block`、`inline-block`、`none`）控制元素显示方式
### 3. CSS 布局
- 浮动与定位
- Flexbox 布局
- Grid 布局
### 4. 响应式设计
- 媒体查询
- 移动端适配
---
## 三、JavaScript / TypeScript
### 1. JavaScript 基础
- **变量类型与基础语法**：条件、循环、函数、类
- **异步编程**：掌握回调函数、`Promise`、`async/await` 等概念
- **DOM 操作**：熟练使用原生 API 进行文档对象模型的操作
- **事件处理**：理解事件冒泡、捕获机制及如何有效地添加和移除事件监听器
### 2. JS 通用对象方法
- **String（字符串）**：查找、替换、分割、子字符串、正则表达式
- **Array（数组）**：添加、删除、合并、排序、遍历
- **Object（对象）**：判断属性、添加修改属性、遍历属性
### 3. TypeScript 核心
- **类型系统**：掌握基本类型、接口、类、枚举、泛型等
- **高级类型**：学习如何使用交叉类型、联合类型、类型保护等高级特性
- **编译选项**：理解 `tsconfig.json` 文件中的配置项（如 `target`、`module`、`strict` 等）
### 4. TS 重点掌握
- **Array 数组方法**：`filter`、`join`、`map`
- **TypeScript 接口**
- **TypeScript 泛型**
---
## 四、Vue 框架
### 1. 环境搭建
- 安装 Node.js 和 npm
- 安装 Vue CLI
- 创建 Vue 项目
### 2. 核心概念
- **数据绑定与事件处理**
- **常用指令**：`v-if`、`v-show`、`v-for`、`v-on`、`v-bind`、`v-model`
- **生命周期钩子**：掌握 Vue 的 8 个生命周期：`beforeCreate`、`created`、`beforeMount`、`mounted`、`beforeUpdate`、`updated`、`beforeUnmount`、`unmounted`
### 3. 组件化开发
- **组件注册与使用**：全局组件、局部组件
- **组件通信**：
  - 父子通信：`props` 向下传递，`$emit` 向上触发事件
  - 跨组件通信：使用 `$root`、`$bus` 或 Vuex（状态管理库）
### 4. Vue 进阶学习
- **路由管理**：
  - 安装与配置
  - 动态路由与嵌套路由
  - 路由守卫
- **状态管理**：
  - `State`：存储全局状态
  - `Getters`：派生状态（类似计算属性）
  - `Mutations`：同步修改状态的方法
  - `Actions`：异步操作（如 API 请求）
  - `Modules`：模块化管理大型应用的状态
- **项目构建工具**：
  - Webpack 的核心概念（入口、输出、加载器、插件）
  - Vite：新一代前端构建工具
### 5. Vue 3 新特性
- **Composition API**：使用 `setup()` 函数替代 Options API
- **响应式原理**：
  - 响应式系统：`ref`、`reactive`、`computed`、`watch`
  - Vue 3 使用 `Proxy` 替代 `Object.defineProperty`，实现更高效的响应式系统
- **内置新组件**：
  - `Teleport`：将组件渲染到 DOM 的任意位置（如模态框）
  - `Suspense`：处理异步组件加载状态
### 6. Vue 性能优化
- **代码懒加载**：使用 `defineAsyncComponent` 或动态导入
- **防抖与节流**：在高频事件（如 `scroll`、`resize`）中优化性能
- **虚拟滚动**：使用 `vue-virtual-scroller` 等库优化长列表渲染
### 7. Vue 测试与部署
- 使用 Vue CLI 或 Vite 构建生产环境包
---

- 配合 GitHub 管理代码
- **核心要求**：必须熟练掌握分支创建、分支切换、代码合并等协同操作

## 六、HTTP 协议
- HTTP 协议工作原理
- HTTP 请求方法

- HTTP 报文组成
- **RESTful API**：理解和使用 REST 原则构建和消费 API

## 七、F12 - 开发者工具详解
- **元素**：检查和实时修改 DOM 结构与 CSS 样式
- **控制台**：执行 JS 代码、查看日志输出与报错信息
- **源代码**：断点调试 JavaScript 代码
- **网络**：分析 HTTP 请求报文、响应报文及加载性能耗时
